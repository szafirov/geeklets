# geeklets
My repository of [geeklets][10].

Makes use of the following commands:

- curl, grep, sed, awk, tr, head, tail, column, date, cal, ps, top, ifconfig, sysctl
- Python [json.tool][6]
- [iCalBuddy][1]
- [fetchmail][2]
- [archey-osx][3]
- [burl][4], can easily be replaced by curl
- [ioreg][5]
